Animation Converter

Dev-only batch tool. Walks every FBX in _refs/_FBX-Animations/MixamoForSimbox/, parses it with Three.js's FBXLoader, applies the same leg-strip + root-motion-strip normalization that animationManager.js uses today, then bundles all clips into a single JSON file at public/holodeck/global_assets/animations/mixamo-simbox.json.

Output bundle is web-native: drop the FBXLoader at runtime, fetch() the JSON, and use THREE.AnimationClip.parse(clip) to rehydrate.
